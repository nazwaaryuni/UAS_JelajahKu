# Task 5: Data Ekstrakurikuler

## Deskripsi
Fitur pelengkap untuk mencatat partisipasi dan capaian siswa dalam kegiatan non-akademik di luar jam pelajaran resmi.

## Kebutuhan (Requirements)
1. **Pembuatan Struktur Tabel**:
   - `extracurriculars`.
2. **Pembuatan Dummy Data (Seeder)**:
   - Generate seeder berisi partisipasi dummy (contoh: Pramuka, PMR, Paskibra) dan nilai predikat (Sangat Baik, Baik, Cukup) untuk siswa.
3. **Pembuatan Fitur**:
   - Modul CRUD bagi Wali Kelas atau Admin untuk memasukkan rekam jejak ekskul siswa.
   - Riwayat ekstrakurikuler di dasbor Siswa.
